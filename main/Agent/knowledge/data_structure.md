# 知识库目录结构

本知识库为农业病虫害知识库，包含多种农作物病虫害的诊断与防治信息。

## 目录概览

| 目录/文件 | 用途说明 |
|----------|---------|
| `agriculture_dat.json` | 农业知识JSON数据库（约6.9MB），包含病虫害问答数据 |
| `LLM_books/` | 农作物病虫害知识文档集合 |

## 使用说明

- **文本检索**：使用 grep 工具在 .txt 文件中搜索关键词
- **JSON数据库检索**：agriculture_dat.json 包含大量病虫害问答数据，可用 grep 搜索
- **按作物查找**：先确定作物类别，再进入对应目录查找具体病害/虫害
- **知识库根目录**：`e:/code/Django/dashboard/Django-dashboard/aiModels/agent/knowledge/`
